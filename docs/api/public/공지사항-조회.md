# 공지사항 조회

## API 정보

- **API명**: 공지사항 조회
- **메서드**: GET
- **URL**: https://api.bithumb.com/v1/notice
- **설명**: 빗썸 공지사항을 조회합니다.

> 예시코드는 JavaScript, Python, JAVA에 한해서만 제공합니다.

## Request

### Query Parameters

| 필드명 | 타입 | 필수 여부 | 설명 | 기본값 |
|--------|------|----------|------|--------|
| count | Number | X | 공지사항 게시물 조회 개수 (기본: 5, 최소: 1, 최대: 20) | 5 |

## Response

### Response Fields

| 필드명 | 타입 | 설명 |
|--------|------|------|
| categories | Array[String] | 공지사항 카테고리 |
| title | String | 공지사항 제목 |
| pc_url | String | 공지사항 URL |
| published_at | String | 공지사항 게시시간<br>포맷: `yyyy-MM-dd hh:mm:ss` |
| modified_at | String | 공지사항 수정시간<br>포맷: `yyyy-MM-dd hh:mm:ss` |

### Response Codes

| 상태코드 | 설명 |
|----------|------|
| 200 | 성공 |
| 400 | 요청 파라미터 오류 |
