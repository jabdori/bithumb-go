# 호가 정보 조회

## API 정보

- **API명**: 호가 정보 조회
- **메서드**: GET
- **URL**: https://api.bithumb.com/v1/orderbook
- **설명**: 특정 마켓의 호가 정보(Orderbook)를 조회합니다.

> 예시코드는 JavaScript, Python, JAVA에 한해서만 제공합니다.

## Request

### Query Parameters

| 필드명 | 타입 | 필수 여부 | 설명 | 기본값 |
|--------|------|----------|------|--------|
| markets | string | O | 마켓 코드 (예: KRW-BTC) | - |

## Response

### Response Fields

| 필드명 | 타입 | 설명 |
|--------|------|------|
| market | String | 마켓 코드 |
| timestamp | Long | 호가 생성 시각 |
| total_ask_size | Double | 호가 매도 총 잔량 |
| total_bid_size | Double | 호가 매수 총 잔량 |
| orderbook_units | List of Objects | 호가 |
| orderbook_units.ask_price | Double | 매도호가 |
| orderbook_units.bid_price | Double | 매수호가 |
| orderbook_units.ask_size | Double | 매도 잔량 |
| orderbook_units.bid_size | Double | 매수 잔량 |

### Response Codes

| 상태코드 | 설명 |
|----------|------|
| 200 | 성공 |
| 400 | 요청 파라미터 오류 |
