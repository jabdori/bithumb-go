# 경보제

## API 정보

- **API명**: 경보제
- **메서드**: GET
- **URL**: https://api.bithumb.com/v1/market/warning
- **설명**: 경보중인 마켓-코인 목록 조회

> 예시코드는 JavaScript, Python, JAVA에 한해서만 제공합니다.

## Request

이 API는 요청 파라미터가 필요하지 않습니다.

## Response

### Response Fields

| 필드명 | 타입 | 설명 |
|--------|------|------|
| market | String | 빗썸에서 제공중인 시장 정보 |
| warning_type | String | 경보 유형<br>- `PRICE_SUDDEN_FLUCTUATION`: 가격 급등락<br>- `TRADING_VOLUME_SUDDEN_FLUCTUATION`: 거래량 급등<br>- `DEPOSIT_AMOUNT_SUDDEN_FLUCTUATION`: 입금량 급등<br>- `PRICE_DIFFERENCE_HIGH`: 가격 차이<br>- `SPECIFIC_ACCOUNT_HIGH_TRANSACTION`: 소수계좌 거래 집중<br>- `EXCHANGE_TRADING_CONCENTRATION`: 거래소 거래 집중 |
| end_date | String | 가상 자산 경보 종료일시(KST 기준)<br>포맷: `yyyy-MM-dd HH:mm:ss` |

### Response Codes

| 상태코드 | 설명 |
|----------|------|
| 200 | 성공 |
| 400 | 요청 파라미터 오류 |
