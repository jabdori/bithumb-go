# 마켓 코드 조회

## API 정보

- **API명**: 마켓 코드 조회
- **메서드**: GET
- **URL**: https://api.bithumb.com/v1/market/all
- **설명**: 빗썸에서 거래 가능한 마켓과 가상자산 정보를 제공합니다.

> 예시코드는 JavaScript, Python, JAVA에 한해서만 제공합니다.

## Request

### Query Parameters

| 필드명 | 타입 | 설명 | 기본값 |
|--------|------|------|--------|
| isDetails | boolean | 유의종목 필드와 같은 상세 정보 노출 여부(선택 파라미터) | false |

### Request Example

#### JavaScript

```javascript
const options = {method: 'GET', headers: {accept: 'application/json'}};

fetch('https://api.bithumb.com/v1/market/all?isDetails=false', options)
  .then(response => response.json())
  .then(response => console.log(response))
  .catch(err => console.error(err));
```

## Response

### Response Fields

| 필드명 | 타입 | 설명 |
|--------|------|------|
| market | String | 빗썸에서 제공중인 시장 정보 |
| korean_name | String | 거래 대상 디지털 자산 한글명 |
| english_name | String | 거래 대상 디지털 자산 영문명 |
| market_warning | String | 유의 종목 여부<br>- `NONE` (해당 사항 없음)<br>- `CAUTION` (거래유의)<br><br>'유의 종목' 지정 여부에 대해 반환하며, '주의 종목' 지정 여부는 [경보제 API](../경보제.md)를 참고하시기 바랍니다. |

### Response Codes

| 상태코드 | 설명 |
|----------|------|
| 200 | 성공 |
| 400 | 요청 파라미터 오류 |
