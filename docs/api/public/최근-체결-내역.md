# 최근 체결 내역

## API 정보

- **API명**: 최근 체결 내역
- **메서드**: GET
- **URL**: https://api.bithumb.com/v1/trades/ticks
- **설명**: 최근 체결 내역을 조회합니다.

> 예시코드는 JavaScript, Python, JAVA에 한해서만 제공합니다.

## Request

### Query Parameters

| 필드명 | 타입 | 필수 여부 | 설명 | 기본값 |
|--------|------|----------|------|--------|
| market | string | O | 마켓 구분 코드 | - |
| to | string | X | 마지막 체결 시각 (exclusive). 비워서 요청시 가장 최근 체결 | - |
| count | int32 | X | 체결 개수(최대 500개까지 요청 가능) | 1 |

## Response

### Response Fields

| 필드명 | 타입 | 설명 |
|--------|------|------|
| market | String | 마켓 구분 코드 |
| trade_date_utc | String | 체결 일자(UTC 기준)<br>포맷: `yyyy-MM-dd` |
| trade_time_utc | String | 체결 시각(UTC 기준)<br>포맷: `HH:mm:ss` |
| timestamp | Long | 체결 타임스탬프 |
| trade_price | Double | 체결 가격 |
| trade_volume | Double | 체결량 |
| prev_closing_price | Double | 전일 종가(UTC 0시 기준) |
| change_price | Double | 변화량 |
| ask_bid | String | 매도/매수 |
| sequential_id | Long | 체결 번호(Unique) |

### Response Codes

| 상태코드 | 설명 |
|----------|------|
| 200 | 성공 |
| 400 | 요청 파라미터 오류 |
