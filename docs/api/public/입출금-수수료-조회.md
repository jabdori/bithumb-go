# 입출금 수수료 조회

## API 정보

- **API명**: 입출금 수수료 조회
- **메서드**: GET
- **URL**: https://api.bithumb.com/v2/fee/inout/{currency}
- **설명**: 빗썸에 상장된 가상자산의 최소 입출금 수량과 입출금 수수료를 제공합니다.

> 예시코드는 JavaScript, Python, JAVA, Shell에 한해서만 제공합니다.

## Request

### Path Parameters

| 필드명 | 타입 | 필수 여부 | 설명 |
|--------|------|----------|------|
| currency | string | O | 화폐를 의미하는 영문 대문자 코드 (`ALL`로 요청 시 전체 응답) |

### Request Example

#### JavaScript

```javascript
const options = {method: 'GET', headers: {accept: 'application/json'}};

fetch('https://api.bithumb.com/v2/fee/inout/ALL', options)
  .then(response => response.json())
  .then(response => console.log(response))
  .catch(err => console.error(err));
```

## Response

### Response Fields

| 필드명 | 타입 | 설명 |
|--------|------|------|
| name | String | 코인명 |
| currency | String | 코인 심볼 |
| networks | Array[Object] | 입출금 네트워크 정보 |
| networks.net_name | String | 입출금 네트워크명 |
| networks.deposit_fee_quantity | NumberString | 입금 수수료 |
| networks.deposit_minimum_quantity | NumberString | 최소 입금 수량 (최소 입금 수량 미만으로 입금할 경우, 수수료 부과) |
| networks.withdraw_fee_quantity | NumberString | 출금 수수료 |
| networks.withdraw_minimum_quantity | NumberString | 최소 출금 수량 |

### Response Codes

| 상태코드 | 설명 |
|----------|------|
| 200 | 성공 |
| 400 | 요청 파라미터 오류 |
