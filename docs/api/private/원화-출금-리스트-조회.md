# View the list of Korean won withdrawals

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/원화-출금-리스트-조회

## 설명

View the list of Korean Won withdrawals.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| state | Withdrawal status
-PROCESSING: In progress
-DONE: Completed
-CANCELED: Cancelled | String |
| uuids | List of withdrawal UUIDs | Array |
| txids | List of withdrawal TXIDs | Array |
| page | Number of pages (default: 1) | Number |
| limit | Number limit (default: 100, max: 100) | Number |
| order_by | Sorting method
-asc: ascending order
-desc: descending order (default) | String |
