# List of allowed withdrawal addresses inquiry

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/출금-허용-주소-리스트-조회

## 설명

View the list of registered withdrawal allowed addresses (addresses that allow withdrawals of 1 million won or more).

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| currency | English capital letter code for currency | String |
| net_type | Withdrawal Network Type | String |
| network_name | Withdrawal network name | String |
| withdraw_address | Withdrawal address | String |
| secondary_address | Secondary withdrawal address (only for required digital assets) | String |
| exchange_name | Withdrawal Exchange Name (English) | String |
| owner_type | Address Owner Customer Type
-personal: personal
-corporation: corporation | String |
| owner_ko_name | Address owner's Korean name | String |
| owner_en_name | English name of address owner | String |
| owner_corp_ko_name | Address Ownership Corporation Korean Name | String |
| owner_corp_en_name | English name of the address-owning corporation | String |
