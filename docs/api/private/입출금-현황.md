# Deposit/Withdrawal Status

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/입출금-현황

## 설명

Check deposit/withdrawal status and block status.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| currency | English capital letter code for currency | String |
| wallet_state | Deposit/Withdrawal Status
-working: Deposits and withdrawals possible
-withdraw_only: Withdrawal only
-deposit_only: Deposit only
-paused: Deposits and withdrawals paused | String |
| block_state | Block status
-normal: normal
-delayed: delayed
-inactive: inactive (under maintenance, etc.) | String |
| block_height | Block height | Integer |
| block_updated_at | Block update time | DateString |
| block_elapsed_minutes | Time elapsed since last block information update (minutes) | Integer |
| net_type | Network type to be specified when requesting deposits or withdrawals | String |
| network_name | Deposit/Withdrawal Network Name | String |
