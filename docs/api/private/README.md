## 문서 형식

각 API 문서는 다음 섹션들을 포함합니다:
- 제목 및 원본 URL
- HTTP Method
- Endpoint
- 설명
- Headers (필요한 경우)
- Query Parameters (필요한 경우)
- Body Parameters (필요한 경우)
- Response 필드
- 예시 코드 (제공되는 경우)

## 생성 방법

이 문서들은 Playwright를 사용하여 빗썸 API 문서 사이트(https://apidocs.bithumb.com/v2.1.5/reference)에서 자동으로 추출되었습니다.

생성 스크립트: `/scripts/extract-private-apis.js`

## 참고사항

- 모든 PRIVATE API는 JWT 토큰 인증이 필요합니다.
- API 키 발급은 빗썸 공식 웹사이트에서 가능합니다.
- BETA로 표시된 API는 아 개발 중인 기능으로, 변경될 수 있습니다.
