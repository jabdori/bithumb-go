# Check the list of Korean won deposits

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/원화-입금-리스트-조회

## 설명

View the list of KRW deposits.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| state | Deposit status
-PROCESSING: In progress
-ACCEPTED: Completed
-CANCELED: Cancelled | String |
| uuids | List of deposit UUIDs | Array |
| txids | List of deposit TXIDs | Array |
| page | Number of pages (default: 1) | Number |
| limit | Number limit (default: 100, limit: 100) | Number |
| order_by | Sorting method
-asc: ascending order
-desc: descending order (default) | String |
