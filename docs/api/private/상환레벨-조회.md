# Repayment level inquiry

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/상환레벨-조회

## 설명

When using Coin Lending (Lending Plus), check your repayment level information.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| total_balance | My total asset valuation | NumberString |
| warranty_balance | My Major Asset Valuation | NumberString |
| lending_amount | Total amount to be repaid in Korean Won equivalent | NumberString |
| margin_level | Repayment level Example) '2.00' *Withdrawal is possible for amounts of 2 or more. | NumberString |
| calculated_at | Repayment level standard date | DateString |
