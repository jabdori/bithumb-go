# Individual deposit inquiry

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/개별-입금-조회

## 설명

Check the deposit details for the deposit by using the deposit UUID.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| currency * | English capital letter code for currency | String |
| uuid | Deposit UUID | String |
| txid | Deposit TXID | String |
