# Withdraw Korean won

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/원화-출금하기

## 설명

Request a withdrawal in Korean Won to your registered withdrawal account.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| amount * | Withdrawal amount | NumberString |
| two_factor_type * | Secondary authentication method
-kakao: Kakao authentication | String |
