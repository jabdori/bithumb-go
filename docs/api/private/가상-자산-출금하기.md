# Withdrawing Virtual Assets

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/디지털-자산-출금하기

## 설명

Request a withdrawal of virtual assets.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| currency * | English capital letter code for currency | String |
| net_type * | Withdrawal network (ex. BTC, DASH) | String |
| amount * | Withdrawal amount | Number |
| address * | Withdrawal address registered in the withdrawal address list | String |
| secondary_address | Secondary withdrawal address (only for required digital assets) | String |
| exchange_name | Withdrawal Exchange Name (English) | String |
| receiver_type | Whether the recipient is an individual or a corporation
-personal: personal
-corporation: corporation | String |
| receiver_ko_name | Recipient's Korean name (Individual: Individual's Korean name, Corporation: Corporation's representative's Korean name) | String |
| receiver_en_name | Recipient's English name (Individual: Individual's English name, Corporation: Corporation's representative's English name) | String |
| receiver_corp_ko_name | Korean name of the corporation (required if the recipient is a corporation) | String |
| receiver_corp_en_name | English name of the corporation (required if the recipient is a corporation) | String |
