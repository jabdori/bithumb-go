# Individual withdrawal inquiry

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/개별-출금-조회

## 설명

Check the withdrawal details for the withdrawal transaction using the withdrawal UUID.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| currency * | English capital letter code for currency | String |
| uuid | Unique ID for withdrawal | String |
| txid | Transaction ID of withdrawal | String |
