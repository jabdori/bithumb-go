# Request to create a deposit address

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/입금-주소-생성-요청

## 설명

Request to create a deposit address.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| currency * | English capital letter code for currency (ex. BTC) | String |
| net_type * | Deposit network (ex. BTC, DASH) | String |
