# Order [BETA]

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/주문하기

## 설명

I would like to request an order.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| market * | Market ID (ex.KRW-BTC) | String |
| side * | Order Type
-bid: buy
-ask: sell | String |
| order_type* | Order type (formerly ord_type)
-limit: limit order
-price: Market order (buy)
-market: Market order (sell) | String |
| price * | Order price (required for limit and market purchases)
ex) If you trade 1,000 KRW per 1 BTC in the KRW-BTC market, the value will be 1000.
ex) If the first sell order price per 1 BTC in the KRW-BTC market is 500 KRW, setting the price to 1,000 when buying at the market price will result in the purchase of 2 BTC. (This may vary depending on the commission or the quantity of the first sell order.) | NumberString |
| volume * | Order quantity (required for limit and market sells) | NumberString |
