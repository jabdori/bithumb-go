# View all accounts

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/전체-계좌-조회

## 설명

View information about the assets you hold.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| currency | English capital letter code for currency | String |
| balance | Orderable amount/quantity | NumberString |
| locked | Amount/quantity tied to the order | NumberString |
| avg_buy_price | Average purchase price | NumberString |
| avg_buy_price_modified | Whether to revise the average purchase price | Boolean |
| unit_currency | Currency based on average price | String |
