# View order list

> **원본 URL**: https://apidocs.bithumb.com/v2.1.5/reference/주문-리스트-조회

## 설명

View your order list.

## Response

| 필드 | 설명 | 타입 |
|------|------|------|
| market | Market ID (ex.KRW-BTC) | String |
| uuids | List of order UUIDs | Array |
| state | Order Status
-wait: Wait for settlement (default)
-watch: Waiting for pre-order
-done: Completed transaction
-cancel: cancel order | String |
| states | List of order statuses
General order(wait, done, cancel) and automatic ordering (watch) cannot be searched together. | Array |
| page | Number of pages (default: 1) | Number |
| limit | Number limit (default: 100, limit: 100) | Number |
| order_by | Sorting method
-asc: ascending order
-desc: descending order (default) | String |
